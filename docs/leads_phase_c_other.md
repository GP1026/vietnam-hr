## 四、调研小结

### 各渠道获客可行性评估

| 渠道 | 可行性 | 获客成本 | 推荐优先级 | 说明 |

| **Zalo 中文求职群** | ★★★★★ | 低 | **最高** | 越南人日常使用 Zalo，中文工作群精准触达目标人群 |
| **vinazhaopin.com 简历库** | ★★★★☆ | 低 | 高 | 可直接浏览会中文越南人简历，免费发布招聘 |
| **job.vnicn.com** | ★★★★☆ | 低 | 高 | 在 HCMC 有实体办公室，可现场拜访 |

| **Telegram 越南招聘群** | ★★★☆☆ | 低 | 中 | 偏中国人在越南找工作，越南本地人使用少 |
| **越南华人论坛** | ★★★☆☆ | 低 | 中 | 主要服务在越中国人，但可发布招聘信息触达中越双语人群 |
| **电鸭社区** | ★★☆☆☆ | 低 | 低 | 偏远程/技术岗，越南求职者数量有限 |

### 核心发现

1. **Zalo 是第一渠道**：越南本地人（包括会中文的越南人）主要用 Zalo 找工作，不用 Telegram。已找到 4 个中文专项 Zalo 群，建议抵达胡志明后立即加入。

2. **中越双语招聘平台已成生态**：vinazhaopin.com 和 job.vnicn.com 都有会中文越南人的简历库，可直接浏览筛选。job.vnicn.com 在 HCMC 第5郡有实体办公室，距第1郡/第4郡很近，值得实地拜访。

3. **中文岗位需求旺盛**：仅 HCMC 就有 400-4900+ 个中文相关岗位在招，说明"会中文的越南人"是一个真实且规模可观的人才池。

4. **Telegram 群偏灰色**：越南相关的 Telegram 招聘群以中文为主，部分混有博彩/灰产信息，需谨慎筛选正规岗位。越南本地人基本不用 Telegram 找工作。

5. **华人论坛价值有限**：越南华人论坛主要服务在越中国人（非越南本地人），但可作为发布招聘信息的补充渠道。

   - 在 vinazhaopin.com 发布长期招聘信息（免费）
